# Workflow für neue Stilfiguren im Projekt Lausberg-Analysewerkstatt

## Ziel
Jede Stilfigur wird so bearbeitet, dass sie am Ende:
- fachlich präzise beschrieben ist
- sauber gegen ähnliche Figuren abgegrenzt ist
- mehrere belastbare Beispiele enthält
- in das feste JSON-Schema überführt ist
- später für Retrieval, Bewertung und Generierung nutzbar ist

## Standardablauf

### 1. Stilfigur festlegen
- Immer zuerst genau eine Stilfigur auswählen
- Optional: eng verwandte Vergleichsfigur mitführen

### 2. Quellenmaterial sichten
- Lausberg-Auszug
- PDF-Seiten
- vorhandene Projektdateien
- Nutzerhinweise
- frühere Korrekturen

### 3. Begriff präzisieren
- genaue Benennung
- Originalbegriffe
- knappe Fachdefinition
- einfache Arbeitsdefinition

### 4. Strukturkern bestimmen
- formales Muster
- semantische Funktion
- typische Signalmerkmale

### 5. Abgrenzung leisten
- ähnliche Figuren benennen
- Unterschiede klar formulieren
- Grenzfälle markieren

### 6. Beispiele sammeln
- klassische Beispiele
- moderne Beispiele
- Predigtbeispiele
- negative Beispiele
- nur Beispiele aufnehmen, die die Figur wirklich tragen

### 7. Wirkung und Einsatz analysieren
- rhetorische Wirkung
- geeignete Rede- oder Predigtstellen
- Zielgruppen
- Grenzen und Fehlgebrauch

### 8. JSON befüllen
- alle Felder nach Schema ausfüllen
- leere Felder nur dann lassen, wenn wirklich noch keine gesicherte Angabe vorliegt
- Konsistenz mit früheren Dateien prüfen

### 9. Qualitätsprüfung
- Ist die Figur sauber definiert?
- Ist sie gegen ähnliche Figuren abgegrenzt?
- Sind die Beispiele wirklich passend?
- Ist der JSON-Block technisch sauber?
- Gibt es unnötige Unschärfen?

### 10. Optional: kreative Anwendung
Nur wenn ausdrücklich gewünscht:
- Predigtbeispiele
- Redevarianten
- moderne Formulierungen
- humorvolle Varianten
- Stilmittelgenerator-Vorlagen

## Standardregel
Analyse vor Kreativität.
Klassifikation vor Formulierung.
JSON vor freiem Text, sobald es um Korpusaufbau geht.
