# MEISTERREDNER Batch 1

Enthalten sind sieben erste, editierbare Einleger plus JSON-Dateien und OCR-Rohtexte.

Enthaltene Figuren:
- Anaphora
- Annominatio
- Antitheton
- Asyndeton
- Communicatio
- Commutatio
- Derivatio

Hinweis:
Diese erste Charge basiert auf den bereits robust ausgelesenen Scan-PDFs. Griechische und mehrsprachige Stellen wurden wo nötig vorsichtig normalisiert. Für eine endgültige Editionsfassung sollten die Originalstellen noch einmal gegen den Scan gegengeprüft werden.
