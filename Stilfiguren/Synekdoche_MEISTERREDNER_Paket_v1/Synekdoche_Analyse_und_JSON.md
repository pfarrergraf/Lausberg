# Synekdoche — Analyse, Korpusfassung und Einleger-Bausteine

## 1. Hauptfigur
- **Deutsch:** Synekdoche
- **Griechisch:** συνεκδοχή
- **Lateinisch:** synecdoche

Im relevanten PDF-Abschnitt ist die Figur als eigener Hauptblock **C) synecdoche (§§ 572–577)** markiert. Für dieses Paket bleibt der Fokus bewusst eng: nur die Hauptfigur Synekdoche und ihr enger Kernbestand.

## 2. Präzise Arbeitsdefinition
Synekdoche ist die Benennung eines Gemeinten durch einen Ausdruck, der zu ihm in einer Zugehörigkeits-, Umfangs- oder Klassenrelation steht: Teil für Ganzes, Ganzes für Teil, Art für Gattung, Gattung für Art oder Singular für Plural bzw. umgekehrt.

### Einfach gesagt
Nicht das eigentlich Gemeinte wird direkt genannt, sondern ein Teil, ein Ganzes, eine Klasse oder eine Zahlform davon.

## 3. Strukturkern
- pars pro toto
- totum pro parte
- species pro genere
- genus pro specie
- singularis pro plurali
- pluralis pro singulari

## 4. Rhetorische Funktion
- Verdichtung
- Konkretisierung
- Verkörperung
- anschauliche Benennung
- poetische und homiletische Prägnanz

## 5. Abgrenzung
### Gegenüber der Metapher
Metapher beruht auf Ähnlichkeit; Synekdoche auf Zugehörigkeit.

### Gegenüber allgemeiner Metonymie
Nicht jede Metonymie ist Synekdoche. Synekdoche ist der enger bestimmte Bereich der Umfangs- und Klassenrelation.

### Gegenüber bloßer Generalisierung
Nicht jede Generalisierung ist schon Synekdoche. Erst die klare Stellvertretungsrelation macht die Figur sauber.

## 6. Sichere klassische Beispiele
- **tectum / tecta** — Dach / Dächer für Haus
- **parietes** — Wände für Gebäude / Haus
- **puppis** — Heck für Schiff
- **frigidus annus** — kaltes Jahr für Winter
- **J’ignore le destin d’une tête si chère.** — Ich kenne das Schicksal eines so teuren Hauptes / einer so teuren Person nicht.
- **quadrupedem** — Vierfüßler für Pferd
- **ferrum** — Eisen für Schwert
- **Romanus** — der Römer für die Römer
